# Evodo

様々な技術を試すためのtodoリスト開発プロジェクト  
シンプルな機能の実装から始めて、段階的に複雑な機能を実装していきたい。  

単純な機能だけを実装して終わりにしたくなかったので、  
進化するTodoリスト(Evolving todo list)をもじってevodoという名前にする😏
